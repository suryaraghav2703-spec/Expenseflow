# ExpenseFlow – Personal Expense Tracker

A modern React-based expense tracking application with budgeting, charts, and history tracking.

---

## Features

- Add, edit, and delete expenses
- Category-based tracking (Bills, Food, Travel, etc.)
- Budget management with warning
- Charts (Bar & Pie)
- Save sessions to history
- LocalStorage persistence
